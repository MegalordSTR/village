# 00-003-01: Advanced Economics Domain Layer

Feature: F003 (Advanced Economics System)

## Goal

Define domain entities and value objects for advanced economics: Currency, Market, Price, TradeRoute, Caravan, Bank, Loan, EconomicCrisis. These types will be used by the simulation systems and provide the foundation for trade, banking, and economic crisis systems.

## Scope Files

- `internal/economy/advanced_types.go` - Core domain types for advanced economics
- `internal/economy/currency.go` - Currency and monetary system types
- `internal/economy/market.go` - Market price dynamics and supply/demand
- `internal/economy/trade.go` - Trade routes, caravans, and negotiation logic
- `internal/economy/banking.go` - Banking, loans, and credit system
- `internal/economy/crisis.go` - Economic crisis detection and effects
- `internal/economy/constants.go` - Add base prices and economic constants

## Dependencies

- 00-001-08: Economic Systems (basic economy simulation)
- 00-002-??: (Any completed workstreams for resource economy) - independent

## Acceptance Criteria

- [ ] Currency types defined (Penny, Shilling, Pound) with metal purity and exchange rates
- [ ] Market type with price history, supply/demand tracking, and price calculation methods
- [ ] TradeRoute and Caravan types with distance, security, and merchant data
- [ ] Bank and Loan types with interest rates, collateral, and payment tracking
- [ ] EconomicCrisis types for inflation, deflation, bank runs, bubbles, and depressions
- [ ] All types have JSON tags for serialization
- [ ] Unit tests for type validation and basic methods
