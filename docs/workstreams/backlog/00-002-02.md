# 00-002-02: Production Chain System

Feature: F002 (Resource Economy System)

## Goal

Implement production recipes and chains for transforming raw materials into processed and advanced goods. Define multi-step production pipelines (wheat → flour → bread, wood → planks → furniture, iron ore → iron → tools/weapons, wool → yarn → cloth → clothing) with input/output requirements, worker skills, building requirements, and time constraints.

## Scope Files

- `internal/economy/recipes.go` - Recipe struct, ResourceRequirement, ResourceOutput, recipe definitions
- `internal/economy/production_chains.go` - Chain definitions, dependency graphs, bottleneck detection
- `internal/economy/calculator.go` - Production calculation algorithm considering worker skill, building condition, quality inheritance
- `internal/economy/recipes_test.go` - Unit tests for recipe validation and production calculations

## Dependencies

- 00-002-01: Resource Definition System (requires resource types)

## Acceptance Criteria

- [ ] Recipe struct with fields: ID, Name, Building, Skill, Inputs[], Outputs[], Time, BaseYield, MaxWorkers
- [ ] ResourceRequirement struct: Type, Quantity, Quality (min)
- [ ] ResourceOutput struct: Type, Quantity, QualityInheritance factor
- [ ] Define at least 10 production recipes covering all 4 chains (food, wood, metal, textile)
- [ ] Production chain visualization helper (returns chain steps for a given output)
- [ ] CalculateProduction function considering worker skills, building condition, input quality
- [ ] Quality inheritance: output quality based on input quality and worker skill
- [ ] Partial production when inputs insufficient
- [ ] Deterministic output given same inputs and RNG seed
- [ ] Unit tests for all recipes and edge cases
- [ ] `go build ./internal/economy` passes
- [ ] `go test ./internal/economy -run TestRecipe` passes with ≥80% coverage
