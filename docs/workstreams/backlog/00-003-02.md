# 00-003-02: Advanced Economics Application Layer

Feature: F003 (Advanced Economics System)

## Goal

Integrate advanced economics domain with the simulation turn processor. Extend the EconomicSystem to handle trade, currency, market prices, banking, and economic crises. Ensure advanced economic updates are processed deterministically each week.

## Scope Files

- `internal/simulation/advanced_economy.go` - Advanced economic system implementing System interface
- `internal/simulation/advanced_economy_test.go` - Unit tests for advanced economy
- `internal/simulation/economy.go` - Extend existing EconomicSystem to incorporate advanced features
- `internal/simulation/economy_test.go` - Update tests for extended functionality
- `internal/simulation/system.go` - Add SystemAdvancedEconomic constant (optional)
- `internal/simulation/turn_processor.go` - Register advanced economic system in processing order

## Dependencies

- 00-003-01: Advanced Economics Domain Layer (provides types)
- 00-001-08: Economic Systems (basic economy simulation)

## Acceptance Criteria

- [ ] AdvancedEconomicSystem type implements System interface
- [ ] EconomicSystem extended with methods: updateMarketPrices, processTrade, updateBanking, checkCrises
- [ ] Advanced economic updates called from EconomicSystem.Update or separate system
- [ ] Market prices updated weekly based on supply/demand and seasonality
- [ ] Trade caravans arrive and depart according to trade routes
- [ ] Banking system updates loans, interest, and deposits weekly
- [ ] Economic crises detected and effects applied
- [ ] All advanced economic systems are deterministic (seed-based RNG)
- [ ] Integration with existing resource consumption and wealth distribution
- [ ] `go build ./internal/simulation` passes
- [ ] `go test ./internal/simulation -run TestAdvancedEconomy` passes with ≥80% coverage
- [ ] `go test ./internal/simulation -run TestEconomic` passes (no regression)
