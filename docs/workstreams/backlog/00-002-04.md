# 00-002-04: Seasonal Economic Cycle

Feature: F002 (Resource Economy System)

## Goal

Implement seasonal patterns affecting agriculture, resource availability, and economic activity. Create a realistic medieval agricultural calendar (planting in spring, growth in summer, harvest in autumn, dormancy in winter) with weather effects on crop yields. Add seasonal resource price fluctuations and winter challenges (increased fuel consumption, no agriculture).

## Scope Files

- `internal/economy/seasons.go` - Season definitions, calendar utilities, seasonal modifiers
- `internal/economy/agriculture.go` - Crop growth stages, planting/harvesting logic, yield calculations
- `internal/economy/weather.go` - Weather patterns (drought, rain, normal) affecting agriculture and resource gathering
- `internal/economy/seasons_test.go` - Unit tests for seasonal logic and agriculture

## Dependencies

- 00-002-01: Resource Definition System (resource types for agriculture)
- 00-002-02: Production Chain System (agriculture production recipes)

## Acceptance Criteria

- [ ] Season enum: Spring, Summer, Autumn, Winter with duration (weeks)
- [ ] Seasonal modifiers: agriculture yield multiplier (0.0 in winter, 1.5 in summer), fuel consumption multiplier (1.0 summer, 2.5 winter)
- [ ] Crop growth stages: planted, growing, ready for harvest, harvested
- [ ] Weather system: drought (0.5x yield), normal (1.0x), rain (1.3x yield but possible flooding)
- [ ] Agricultural calendar: planting window (spring weeks 1-4), harvest window (autumn weeks 8-12)
- [ ] Winter challenges: no agriculture possible, increased wood consumption for heating
- [ ] Seasonal price fluctuations: food cheap after harvest, expensive in spring
- [ ] Storage preparation indicators for winter (recommended stockpiles)
- [ ] Integration with existing Environment system (extends temperature, rainfall)
- [ ] Unit tests covering all seasons and weather scenarios
- [ ] `go build ./internal/economy` passes
- [ ] `go test ./internal/economy -run TestSeason` passes with ≥80% coverage
