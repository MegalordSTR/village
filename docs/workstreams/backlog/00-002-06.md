# 00-002-06: Resource Scarcity & Substitution

Feature: F002 (Resource Economy System)

## Goal

Implement resource scarcity mechanics: mines deplete over time, forests regrow slowly, and players must find alternative solutions when preferred resources are unavailable. Create substitution system allowing alternative materials with efficiency penalties, and scavenging for limited resources in the environment.

## Scope Files

- `internal/economy/scarcity.go` - Scarcity calculations, resource availability modifiers
- `internal/economy/substitution.go` - Substitution rules, alternative materials, efficiency penalties
- `internal/economy/exhaustion.go` - Resource exhaustion (mine depletion, forest health), regeneration rates
- `internal/economy/scarcity_test.go` - Unit tests for scarcity and substitution logic

## Dependencies

- 00-002-01: Resource Definition System (resource types)
- 00-002-02: Production Chain System (alternative recipes)

## Acceptance Criteria

- [ ] Resource exhaustion: mines have finite ore, depletion reduces yield over time
- [ ] Forest health: logging reduces forest health, regrowth over years (10% per year)
- [ ] Scarcity modifiers affect production yield (rich mine: 1.5x, depleted: 0.3x)
- [ ] Substitution system: define alternative materials for recipes (stone instead of wood for buildings)
- [ ] Efficiency penalties for substitutions (stone building: 2x construction time, 0.8x durability)
- [ ] Scavenging: find limited resources in environment (abandoned mines, fallen trees)
- [ ] Strategic trade-offs: use scarce resource now vs. save for future critical need
- [ ] Resource discovery events (new mine vein, abundant forest patch)
- [ ] Integration with production system to apply scarcity modifiers
- [ ] Unit tests covering exhaustion, substitution, and scarcity scenarios
- [ ] `go build ./internal/economy` passes
- [ ] `go test ./internal/economy -run TestScarcity` passes with ≥80% coverage
