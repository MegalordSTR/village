# 00-003-04: Advanced Economics Presentation Layer

Feature: F003 (Advanced Economics System)

## Goal

Provide frontend interface for advanced economics: market dashboard, trade interface, banking interface. Extend backend API to expose economic data and actions. Connect UI components to simulation state.

## Scope Files

- `frontend/src/app/services/economy.service.ts` - Angular service for economic API calls
- `frontend/src/app/components/market-dashboard/` - Market price charts and supply/demand visualization
- `frontend/src/app/components/trade-interface/` - Caravan arrivals, trade negotiation UI
- `frontend/src/app/components/banking-interface/` - Loan applications, deposit/withdrawal, account summary
- `cmd/village/main.go` - Register economic API routes
- `internal/api/economy.go` - REST endpoints for economic data
- `internal/api/economy_test.go` - API tests

## Dependencies

- 00-003-02: Advanced Economics Application Layer (simulation integration)
- 00-003-03: Advanced Economics Infrastructure Layer (state serialization)

## Acceptance Criteria

- [ ] REST API endpoints: GET /api/economy/market, GET /api/economy/trade, POST /api/economy/trade/negotiate, GET /api/economy/bank, POST /api/economy/bank/loan
- [ ] Angular service with methods for fetching economic data
- [ ] Market dashboard component shows current prices, trends, and supply/demand graphs
- [ ] Trade interface displays caravan arrivals and allows negotiation
- [ ] Banking interface shows account summary and loan applications
- [ ] UI components update when simulation advances (turn processing)
- [ ] API tests cover all endpoints with ≥80% coverage
- [ ] `go build ./cmd/village` passes
- [ ] `go test ./internal/api` passes
- [ ] `npm run build --prefix frontend` passes without errors
