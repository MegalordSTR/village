# 00-003-05: Advanced Economics Integration Tests

Feature: F003 (Advanced Economics System)

## Goal

Validate end-to-end integration of advanced economics with simulation systems. Test economic balance, crisis scenarios, trade profitability, and deterministic behavior across multiple turns.

## Scope Files

- `internal/simulation/integration_test.go` - Integration tests for advanced economy within simulation
- `internal/economy/integration_test.go` - Integration tests for economic domain logic
- `test/e2e/advanced_economy_test.go` - End-to-end tests covering full economic cycles

## Dependencies

- 00-003-01: Domain Layer
- 00-003-02: Application Layer
- 00-003-03: Infrastructure Layer
- 00-003-04: Presentation Layer (optional, for API tests)

## Acceptance Criteria

- [ ] Integration tests simulate 52 weeks (1 year) of advanced economic activity
- [ ] Trade cycles: production → pricing → trade → profit validation
- [ ] Banking lifecycle: deposit → loan → interest → repayment → withdrawal
- [ ] Economic crisis scenarios: inflation, bank run, bubble burst, recovery
- [ ] Deterministic behavior: same seed produces identical economic outcomes
- [ ] Balance validation: no infinite money glitches, prices stay within reasonable bounds
- [ ] Performance: economic updates complete within 100ms per week for full system
- [ ] `go test ./internal/simulation -run Integration` passes
- [ ] `go test ./internal/economy -run Integration` passes
