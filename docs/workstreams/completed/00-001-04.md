# 00-001-04: Simulation Core Framework

Feature: F001 (Village Simulation Core)

## Goal

Create the core simulation framework: System interface and TurnProcessor that orchestrates weekly turn processing in deterministic order.

## Scope Files

- `internal/simulation/system.go` - System interface definition
- `internal/simulation/turn_processor.go` - TurnProcessor with ordered system execution
- `internal/simulation/turn_processor_test.go` - Unit tests for turn processing

## Dependencies

- 00-001-02: Game State Model
- 00-001-03: Deterministic Random System

## Acceptance Criteria

- [ ] System interface with Update(week int, state *GameState, rng *rand.Rand) []Event
- [ ] TurnProcessor struct managing ordered list of systems
- [ ] ProcessWeek method that calls each system in order
- [ ] Event collection and addition to GameState history
- [ ] Deterministic execution order (environment → production → social → economic → events)
- [ ] Unit tests verify each system is called in order
- [ ] `go test ./internal/simulation -run TestTurnProcessor` passes with ≥80% coverage
