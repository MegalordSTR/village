# 00-001-03: Deterministic Random System

Feature: F001 (Village Simulation Core)

## Goal

Implement a deterministic random number generator based on a seed, with serialization support to ensure identical game state reproduction.

## Scope Files

- `internal/simulation/rng.go` - Seeded RNG implementation
- `internal/simulation/rng_test.go` - Determinism tests
- `internal/simulation/serialization.go` - Seed save/load functions

## Dependencies

- 00-001-02: Game State Model (needs GameState to store seed)

- [ ] Integration with GameState for seed storage
