# Release v0.1.0 - Resource Economy System

**Release Date:** 2026-03-08  
**Tag:** v0.1.0  
**Previous Version:** None (initial release)

## Overview

This release introduces the Resource Economy System (F002), a comprehensive medieval economy simulation for the Village game. The system models resource production, storage, seasonal variations, quality tiers, and market dynamics.

## Features

### 1. Resource Definition System (00-002-01)
- 15 core resource types: grain, vegetables, wood, stone, iron ore, wool, flour, bread, planks, iron, cloth, tools, furniture, weapons, clothing
- Resource categorization: Raw (6), Processed (5), Advanced (4)
- Quality tiers: Poor, Normal, Good, Excellent, Masterwork
- Spoilage rates for perishable goods
- Type validation and category lookup functions

### 2. Production Chain System (00-002-02)
- Production building types: Mill, Bakery, Sawmill, Forge, Workshop, Tailor
- Conversion recipes with input/output ratios
- Production time modeling
- Building capacity limits
- Worker assignment logic

### 3. Inventory & Storage System (00-002-03)
- Storage building types: Granary, Warehouse, Stockpile
- Spatial organization with slot-based storage
- Capacity limits per resource type
- Inventory management API
- Transfer operations between storage locations

### 4. Seasonal Economic Cycle (00-002-04)
- Four seasons: Spring, Summer, Autumn, Winter
- Seasonal price multipliers (e.g., food cheaper after harvest)
- Weather impact on production yields
- Seasonal resource availability
- Time progression system

### 5. Craftsmanship & Quality System (00-002-05)
- Quality tiers affecting market value
- Craftsmanship skill levels (Novice to Master)
- Quality improvement through skill progression
- Random quality outcomes with skill bias
- Quality-based price multipliers

### 6. Resource Scarcity & Substitution (00-002-06)
- Scarcity tracking per resource
- Price adjustment based on supply/demand
- Substitution logic (e.g., use barley when wheat is scarce)
- Alternative recipe support
- Market equilibrium simulation

### 7. UI Integration (Angular) (00-002-07)
- Angular components for resource management
- Resource grid display with filtering
- Production chain visualization
- Inventory management interface
- Quality and scarcity indicators

### 8. Deployment Setup (00-002-08)
- Docker Compose configuration for backend, frontend, and PostgreSQL
- Multi-stage Dockerfile for Go backend and Angular frontend
- Makefile with build, test, run, and deploy targets
- CI/CD pipeline (GitHub Actions) for automated testing and deployment
- Production deployment script

4. Run `docker-compose up -d` to start services
5. Access the application at http://localhost:80
